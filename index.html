HOMELAB
✖

A SERVER RUNNING DIFFERENT SERVICES

I built myself a cheap server out of old computer parts to run some servers for me and my family. The server runs on Proxmox and I installed and setup a few different services I use regularly either in VMs or LXCs. The server runs HomeAssistant for home automation, Nextcloud for storing personal files, OnlyOffice to edit my personal files in the browser, Vaultwarden to store my passwords, Apache to host this website and Adguard Home to block ads and trackers.
